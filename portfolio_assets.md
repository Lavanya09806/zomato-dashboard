## 📄 Resume-Ready Project Description

**Zomato Restaurant Analytics & Revenue Intelligence Dashboard**  
*Data Scientist / Business Analyst Project*

*   **Objective:** Developed an end-to-end analytics platform to help restaurant owners optimize performance using Zomato’s multi-dimensional dataset (10k+ records).
*   **Data Engineering:** Built a robust cleaning pipeline in Python/Pandas to handle missing values, normalize currency, and engineer features like **Popularity Score** and **Revenue Potential Score**.
*   **Visualization:** Created a multi-page interactive dashboard using **Streamlit** and **Plotly**, featuring KPI tracking, heatmaps for location density, and trend analysis for 20+ cuisines.
*   **Predictive Modeling:** Implemented a **Random Forest Regressor** to predict success ratings of new restaurant concepts with features including cost, location, and service type.
*   **Business Impact:** Generated automated insights identifying "Hidden Gems" (high-rating, low-cost) and underserved market gaps for potential restaurant investors.
